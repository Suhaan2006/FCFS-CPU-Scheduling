# FCFS-CPU-Scheduling
Program to implement First Come First Serve (FCFS) CPU scheduling using C programming language.
And we shall also be calculating calculate:
Arrival Time (AT): The time at which a process enters the ready queue.
Burst Time (BT): The total CPU time required by a process to complete execution.
Completion Time (CT): The time at which a process finishes its execution.
Turnaround Time (TAT): The total time taken from arrival to completion of a process.
Waiting Time (WT): The total time a process spends waiting in the ready queue.
